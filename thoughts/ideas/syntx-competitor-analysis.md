# SyntX.ai - Competitor Analysis

Контекст: Исследование AI-агрегатора SyntX.ai как ориентир для масштабирования GiftMixer

## Что такое SyntX
All-in-one AI агрегатор. 90+ нейросетей под одной подпиской ($10-30/мес).
Web dashboard + Telegram bot. Конкурирует с Poe, Perplexity.

**Эволюция:** Telegram bot (2024) -> Web "Studio" (2025) -> Desktop competitor (2026)

## Бизнес-модель
- Кредитная система (pay-per-use) + подписки
- Starter: ~$10-15/мес, Creator: ~$30/мес
- Ценностное предложение: $240/мес по отдельности = $30/мес через SyntX
- Affiliate program: recurring комиссии (LTV-based)
- Стратегия "land grab": агрессивный прайсинг для захвата рынка

## Модели внутри
- Видео: Veo 3, Sora Pro, Kling, Hailuo
- Изображения: Flux, Midjourney v6.1/v7, Recraft v3, DALL-E
- Текст: Gemini 2.5 Pro, Claude 3.5, DeepSeek-V3, GPT-4o
- Музыка: Suno v5

## Academy (образовательный контент)
Раздел с гайдами по каждой модели. Цель: retention, SEO, экспертное позиционирование.

## Уроки для GiftMixer
1. Telegram-first -> Web expansion (наш путь)
2. Academy для retention (гайды внутри приложения)
3. Referral в Telegram (invite + бонусные кредиты)
4. Быстрая интеграция моделей = трафик

## Критическое различие
SyntX = горизонтальный агрегатор (конкурирует ценой).
GiftMixer = вертикальный продукт (конкурирует уникальностью результата).
Не копировать модель 1:1!
