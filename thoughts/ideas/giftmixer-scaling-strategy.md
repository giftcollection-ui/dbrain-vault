# GiftMixer Scaling Strategy
Дата: 2026-03-14
Контекст: На основе анализа SyntX.ai + текущих возможностей платформы
---

## Текущая позиция
- Telegram Mini App с AI-генерацией стикеров, крафтов, постеров, cinema
- 10+ прямых провайдеров (FAL.ai, Google AI, Lovable, OpenRouter)
- 30+ моделей через Krea.ai proxy
- Lightning credits система
- 6 уровней крафтинга

## Фаза 1: Вертикальная доминация (текущая)
- Стикеры, крафты, постеры — лучший в Telegram
- Cinema pipeline как premium-фича
- Задача: довести техническую часть до production-ready

## Фаза 2: Расширение форматов
- Видео для соцсетей (TikTok/Reels/Stories) — Cinema pipeline уже работает
- Анимации (Lottie/TGS) — pipeline есть
- Музыка к видео (через FAL.ai MiniMax Music / Suno)
- Больше моделей из Krea proxy активно использовать

## Фаза 3: Горизонтальное расширение
- Не только подарки -> любые стикеры, аватары, мемы
- Не только Telegram -> WhatsApp стикеры, Discord
- "Студия" для контент-мейкеров
- Веб-версия (desktop)

## Фаза 4: Платформа
- API для сторонних разработчиков
- Маркетплейс стилей и шаблонов
- White-label решения для бизнеса

## Принципы
- НЕ становиться generic агрегатором (гонка на дно по цене)
- Максимум 5-7 моделей идеально настроенных под задачу, а не 90
- Конкурировать опытом и уникальностью, не ценой
- Админ-панель для управления моделями без деплоя

## Что взять от SyntX
- Academy / обучающий контент внутри приложения
- Referral-механика в Telegram (invite links + бонусные кредиты)
- Быстрая интеграция новых моделей как маркетинговый хук
